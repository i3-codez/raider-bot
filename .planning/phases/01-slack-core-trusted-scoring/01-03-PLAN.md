## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Slack slash command user -> `/raid` handler | A human is attempting to create operational work in Slack |
| Modal submission -> database insert | Untrusted user-provided URL, client name, platform, and datetime become canonical raid data |
| Runtime -> Slack Web API | The bot publishes a visible staff-facing message into the raid channel |
