## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Slack reaction events -> scoring service | Untrusted event delivery can be duplicated, reordered, or malformed |
| Scoring service -> audit tables | Canonical points and audit evidence are written here |
| Timing correction CLI -> database and Slack message update | A manual correction can rewrite score interpretation if not logged carefully |
